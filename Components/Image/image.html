Image
Responsive Image
Responsive Image Components are used to display Images on websites, which are screen responsive. They expand upto the width of the container they are in. To use this image component, name the class as responsive-image for your image element.
Rounded Image
Round Images are the circular, which can be used for representation of profiles in a higher dimensions. To use this component, mention a class as rounded-image for your image.
